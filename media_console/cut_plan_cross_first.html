Cut Sequence

  1. ① cross-cut Full 48" sheet at 55½" from one end
    → Top slab 48"×55½"  ·  Bottom slab 48"×40½" (remaining)
  2. ② rip From top slab (48"×55½"):
    Set fence to 15" → A
    Set fence to 15" → B
    Set fence to 4" → C
    Remainder (≈14") is offcut
  3. ③ cross-cut From bottom slab at 15"
    → D/E strip 48"×15"
  4. ④ cross-cut From remainder at 14¼"
    → F/G strip 48"×14¼"  ·  48"×≈11¼" offcut
  5. ⑤ rip From D/E strip (48"×15"):
    Set fence to 16¼" → D
    Set fence to 16¼" → E
    Remainder (≈15½") is offcut
  6. ⑥ rip From F/G strip (48"×14¼"):
    Set fence to 14¾" → F
    Set fence to 14¾" → G
    Remainder (≈18½") is offcut

Tip: because the fence controls piece width, kerf always eats into the offcut side — no need to calculate or know the blade thickness.

Trade-offs vs. Rip-first Plan

Benefits
  • No kerf width assumption
  • Fence setting = piece size, always
  • Shorter pieces easier to handle on table saw
  • 48"×≈11¼" offcut is wide and usable
Drawbacks
  • No single long offcut for taller pieces
  • More cross-cuts (3 vs. 1)
  • Offcut widths are approximate (kerf-dependent)
